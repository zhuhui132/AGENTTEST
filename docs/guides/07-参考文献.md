# 📚 参考资料

以下资源有助于进一步理解和扩展 AgentTest 相关实现。根据需求选择查阅。

## 官方文档

1. **Python AsyncIO 官方指南** – https://docs.python.org/3/library/asyncio.html
2. **pytest 官方文档** – https://docs.pytest.org/
3. **FastAPI Documentation**（如需构建 API 接口） – https://fastapi.tiangolo.com/
4. **redis-py** – https://redis.readthedocs.io/ （扩展记忆系统时可参考）

## 设计与架构

1. Russell, S., Norvig, P. *Artificial Intelligence: A Modern Approach*（4th） – Agent 概念与决策模型
2. Wooldridge, M. *An Introduction to MultiAgent Systems* – 多 Agent 协作背景
3. Microsoft Azure. *Designing Distributed Systems for Cloud* – 微服务与事件驱动实践

## LLM 与 Agent

1. OpenAI. *Function Calling & Tool Use* – https://platform.openai.com/docs/guides/function-calling
2. LangChain. *Agents Overview* – https://python.langchain.com/docs/modules/agents/
3. Gao, S. et al. *Retrieval-Augmented Generation for Knowledge-Intensive NLP* – arXiv:2005.11401

## 测试与质量

1. ISO/IEC/IEEE 29119 – 软件测试流程与文档标准
2. Fowler, M. *Continuous Integration* – 持续集成实践
3. Google. *Testing on the Toilet* 系列博客 – 单元测试与代码质量心得
