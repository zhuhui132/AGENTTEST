# 🤖 Agent系统文档

一个基于大语言模型的智能Agent系统，具备记忆管理、RAG检索、工具调用等核心功能。

## 🚀 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行Agent
python -c "
from src import IntelligentAgent
agent = IntelligentAgent()
response = await agent.process_message('你好，请介绍一下自己')
print(response.content)
"

# 运行测试
pytest tests/
```

## 📋 目录说明

### 🔧 源代码结构
- **core/**: 提供项目的基础设施，包括配置管理、类型定义、异常处理等
- **agents/**: 智能Agent的核心实现，包含规划、执行和监督功能
- **llm/**: 大语言模型的统一接口，支持多种LLM提供商
- **memory/**: 多层次的记忆系统，支持情景记忆、语义记忆和工作记忆
- **rag/**: 检索增强生成模块，提供高效的文档检索和知识注入
- **utils/**: 通用工具模块，提供辅助功能和常用操作
- **ml/**: 机器学习相关功能，支持模型训练和评估

### 🧪 测试体系
- **unit/**: 单元测试，确保每个模块的正确性
- **integration/**: 集成测试，验证模块间的协作
- **e2e/**: 端到端测试，模拟真实使用场景
- **performance/**: 性能测试，监控系统的性能指标

### 📚 文档体系
- **getting-started/**: 面向新用户的快速入门文档，帮助快速上手
- **development/**: 开发指南文档，提供实际案例和开发流程
- **guides/**: 最佳实践文档，包含详细的技术说明和开发规范
- **api/**: API接口文档，提供完整的接口说明和示例
- **knowledge/**: AI技术知识库，涵盖相关技术领域的知识

### ⚙️ 配置与部署
- **config/**: 多环境配置文件，支持开发、测试和生产环境
- **scripts/**: 自动化脚本，简化常见操作
- **Docker**: 容器化配置，支持快速部署
- **GitHub**: CI/CD配置，自动化测试和部署流程
