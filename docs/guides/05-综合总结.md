## 2. 近期风险与注意事项

1. **文档真实度**：历史版本大量假设内容，仍可能存在遗漏；使用前请以源码为准。
2. **工具调用**：默认仅注册计算器示例，其余公共工具需要手动接入。
3. **异步生态依赖**：若在已有事件循环内同步调用工具，会触发 `ToolError`，需提前规划调用方式。
4. **缺省 LLM**：`BaseLLM` 尚未关联实际服务，开发者应根据需要实现自定义类。
