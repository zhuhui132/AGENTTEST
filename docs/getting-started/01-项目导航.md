## 4. Quick Start <a id="quick-start"></a>

```bash
# 1. 创建虚拟环境并安装依赖
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 2. 运行最小示例
python examples/quickstart.py

# 3. 可选：执行质量自检（需 dev 依赖）
pip install -r requirements-dev.txt
pytest -q
```

示例脚本会：
1. 初始化 `IntelligentAgent`
2. 分别以纯字符串与 `AgentMessage` 发送两条请求
3. 输出回复内容、置信度以及最新状态

进阶示例：`examples/agent_usage_example.py`（结合记忆/RAG/工具的多轮对话），`examples/metrics_demo.py`（指标采集与评估）。
