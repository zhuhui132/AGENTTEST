# 📖 术语表

| 术语 | 英文 | 说明 |
| --- | --- | --- |
| 智能体 | Agent | 能够接收消息、访问记忆/工具并返回响应的软件实体。本项目中的核心实现为 `IntelligentAgent`。 |
| 同步包装器 | Agent Wrapper | 将异步 Agent 封装为同步调用接口的类，便于脚本或旧系统集成。 |
| 记忆系统 | Memory System | 负责存储、检索和维护 `MemoryItem`，支持重要性、时间衰减等策略。 |
| RAG | Retrieval-Augmented Generation | 检索增强生成：先查找相关文档，再将结果注入生成阶段，提高回答准确度。 |
| 工具系统 | Tool System | 管理可调用的外部能力（HTTP、计算、查询等），提供注册、参数校验、执行和健康检查。 |
| 上下文管理 | Context Manager | 负责对话历史截断、摘要、意图推断与实体提取，供 Agent 构造提示。 |
| LLM | Large Language Model | 大语言模型；在本仓库中以 `BaseLLM` 抽象表示，需自行接入具体实现。 |
| ToolResult | Tool Result | 工具执行返回的数据结构，包含是否成功、结果或错误、执行耗时等。 |
| 配置管理器 | ConfigManager | 统一加载 YAML/JSON 配置，支持缓存与环境变量覆盖。 |
| 异步/同步切换 | Async↔Sync | 异步函数在已有事件循环中不能直接调用 `asyncio.run`；需要使用 `await` 或提供同步包装。 |

> 若需新增术语，请在 PR 中更新本表，并同步相关文档引用。
